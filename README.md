# scroll

A library for Dart developers. It is awesome.

## Usage

A simple usage example:

    import 'package:scroll/scroll.dart';

    main() {
       new Scroll('nav.navbar ul.nav li a', duration:1000, easing: Easing.EaseInOutElastic, offsetOptions: {
       'top': -103.8
       });
    }
